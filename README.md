## Hi, I'm Tharun 👋

Senior DevOps engineer. I build and operate the delivery platform that ships everything else: Azure DevOps pipelines, AKS clusters running Argo CD, Terraform for the Azure footprint, and a lot of PowerShell glue to keep the rough edges smooth.

### What I work on

- **Platform engineering** — self-service pipelines, shared Helm charts, golden-path templates so product teams don't re-invent CI/CD each sprint
- **GitOps on AKS** — Argo CD app-of-apps, progressive delivery, OPA Gatekeeper guardrails, KEDA for event-driven autoscaling
- **Infrastructure-as-code** — reusable Terraform modules for the Azure services I use most (AKS, VNet, Key Vault, Storage, ACR), multi-env stacks with remote state
- **Day-2 tooling** — PowerShell and Python utilities for pipeline health, cost reporting, compliance auditing, right-sizing
